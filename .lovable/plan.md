### How you turn it on for a client

1. In Lovable, open **More → Agent integrations**, enable this app's server.
2. In Claude Desktop / ChatGPT / Cursor, add a remote MCP server pointing at:
   - `https://zooga-brain-hub.lovable.app/mcp`
3. The client discovers OAuth via `/.well-known/oauth-protected-resource`, dynamically registers, PKCE flow → your Hebrew consent screen → tokens issued by Supabase → tools appear.
